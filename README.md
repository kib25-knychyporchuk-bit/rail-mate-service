# 🚆 Rail Mate

> **«Подорожуйте з легкістю»** 
> Найзручніший спосіб знайти та придбати квитки на потяги по всій Україні без зайвого клопоту.

**Rail Mate** — це навчальний статичний веб-проєкт, розроблений у рамках студентської практики. Сервіс імітує роботу платформи для пошуку та бронювання залізничних квитків, надаючи користувачам інтуїтивно зрозумілий інтерфейс для планування поїздок.

---

## ✨ Основний функціонал
* **Головний екран та пошук:** Зручна форма для вибору станції відправлення, прибуття, дати та кількості пасажирів.
* **Популярні напрямки:** Блок із рекомендованими маршрутами (Київ – Львів, Одеса, Харків, Дніпро) та актуальними цінами.
* **Імітація бази даних:** Підвантаження списку доступних потягів із локальних JSON-файлів.
* **Інтерактивна взаємодія:** Імітація вибору місць у вагоні та процесу оформлення квитка.

---

## Команда та розподіл ролей
Проєкт розроблено командою з двох осіб. Робота велася у відповідних гілках репозиторію (`frontend` та `backend`):

* **Учасник 1 (Frontend & UI):** Розробка макетів, верстка головної сторінки, карток напрямків, форми пошуку та загального дизайну інтерфейсу.
* **Учасник 2 (Backend & Data):** Створення JSON-структури даних, написання JavaScript-логіки для обробки пошуку, кліків користувача та імітації процесу бронювання.
